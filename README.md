# Sales Analytics Project – Northwind Dataset

## Project Overview
This project analyzes sales data using the Northwind dataset to extract business insights and build an interactive dashboard.

## Tools Used
- SQL (MySQL)
- Excel (Data Cleaning & EDA)
- Power BI (Dashboard)
- GitHub (Project Repository)

## Project Steps
1. Data Extraction using SQL
2. Data Cleaning in Excel
3. Exploratory Data Analysis (EDA)
4. Dashboard Creation in Power BI
5. Business Insights & Recommendations

## Key Insights
- Sales increased year by year.
- USA has the highest number of customers.
- Top-selling products contribute to majority revenue.
- Some customers are high-value repeat customers.
- Delivery performance is mostly on-time.
